# Getting Started with JobSpy

Welcome to JobSpy! This guide will help you get started with the application in just a few minutes.

## What is JobSpy?

JobSpy is a comprehensive job search platform that aggregates job listings from multiple sources including LinkedIn, Indeed, Glassdoor, and more. It provides advanced search, filtering, saved jobs, and alert features to help you find your perfect job.

## Quick Start (5 minutes)

### 1. Create an Account

1. Visit [https://yourdomain.com](https://yourdomain.com)
2. Click "Sign Up" in the top right corner
3. Enter your email address and create a password
4. Click "Create Account"
5. Check your email for a verification link and click it

### 2. Search for Jobs

1. Go to the "Search" page
2. Enter a job title (e.g., "Python Developer")
3. Enter a location (e.g., "San Francisco, CA")
4. Click "Search"
5. Browse the results

### 3. Save Your Favorite Jobs

1. Click the heart icon on any job card to save it
2. View your saved jobs by clicking "Saved Jobs" in the navigation menu

### 4. Set Up Job Alerts

1. Go to "Alerts" in the navigation menu
2. Click "Create New Alert"
3. Enter your search criteria
4. Choose how often you want to be notified (hourly, daily, weekly)
5. Click "Create Alert"
6. You'll receive email notifications when new jobs match your criteria

## Key Features

### Advanced Search

Search for jobs using multiple criteria:
- **Job Title**: The position you're looking for
- **Location**: City, state, or remote
- **Job Type**: Full-time, part-time, internship, contract
- **Salary Range**: Minimum and maximum salary
- **Remote**: Filter for remote positions
- **Distance**: Search within a specific radius

### Filtering

Refine your search results:
- Filter by company
- Filter by job type
- Filter by salary range
- Filter by posting date
- Sort by relevance, date, or salary

### Saved Jobs

Keep track of jobs you're interested in:
- Save jobs for later review
- Add notes to saved jobs
- Export saved jobs to CSV or Excel
- Delete jobs from your saved list

### Job Alerts

Get notified about new job opportunities:
- Create alerts for specific searches
- Choose notification frequency
- Receive email notifications
- Manage multiple alerts
- Pause or delete alerts anytime

### Export Data

Download your job data:
- Export to CSV format
- Export to Excel format
- Export to JSON format
- Include custom fields

## Troubleshooting

### I'm not receiving email notifications

1. Check your spam/junk folder
2. Verify your email address in settings
3. Make sure your alerts are active
4. Check your alert notification frequency

### Search results are empty

1. Try a broader search term
2. Remove location filters
3. Expand the distance radius
4. Check if the job type filter is too restrictive

### I can't save a job

1. Make sure you're logged in
2. Try refreshing the page
3. Check your browser's local storage
4. Contact support if the issue persists

### My saved jobs disappeared

1. Check if you're logged into the correct account
2. Try refreshing the page
3. Check your browser's cache
4. Contact support if the issue persists
